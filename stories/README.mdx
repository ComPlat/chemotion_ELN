# Chemstrap

## Chemstrap is the design system for Chemotion ELN.

Based on Bootstrap, built for complex desktop applications in chemistry, life sciences, and materials sciences.

Chemstrap provides UI components for domain-specific workflows:

- planning and collaboration
- collection and organisation
- analysis and standardisation
- import and export
- sharing and publication

To structure this documentation we use the [Atomic Design terminology](https://atomicdesign.bradfrost.com/table-of-contents/).

1. subatomic - design tokens: colour, type, spacing, elevation.
2. atoms - single UI elements with one clear role.
3. molecules - small, reusable groups of atoms.
4. organisms - full feature sections built from molecules and atoms.
